# French quotation mark

French quotation mark is used to replace the default quotation mark by the french one.

## Syntaxe

```syntaxe
"<text>"
```

| Attributs | Value Type | Required | Description           |
| --------- | ---------- | -------- | --------------------- |
| text      | text       | Required | The text to be quoted |
