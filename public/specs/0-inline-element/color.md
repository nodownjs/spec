# Unicode

The color element displays the color specified left to the text.

## Syntaxe

```syntaxe
`<color>`
```

| Attributs | Value Type        | Required | Description |
| --------- | ----------------- | -------- | ----------- |
| color     | hex or rgb or hsl | Required | The color   |

## Exemples

### With HEX

```
`#F49FFF`
```

### With short HEX

```
`#F49`
```

### With RGB

```
`rgb(244, 159, 255)`
```

### With RGBa

```
`rgba(244, 159, 255, 0.5)`
```

### With HSL

```
`hsl(300, 100%, 80%)`
```

### With HSLa

```
`hsla(300, 100%, 80%, 0.5)`
```
