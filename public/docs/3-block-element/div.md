## Exemples

### Basic new div

```
Im in the first div
------
Im in the second div
```

### New horizontal div

```
Im in the first div
======
I'm in the horizontal div
```

### New horizontal div with sub-div

```
Im in the first div
======
I'm in the horizontal div, the first sub-div
===
I'm in the horizontal div, the second sub-div
```
